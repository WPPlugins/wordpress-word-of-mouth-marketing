== Description ==

WordPress Word of Mouth Marketing Plugin, WP-WOMM for short, is a WordPress 2.0+ plugin automatically inserts a link to share and tell a friend about the post or page as well as provide a link for visitors to subscribe to the RSS feed. 

Features:

* Customizable display
* Provides link for visitors to subscribe to RSS feed
* Displays Add This and Tell A Friend links at the bottom of each post and page


== Installation ==

1. Upload `wp-womm.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'settings' --> 'WP-WOMM' and click the 'click here to activate plug' button

== Frequently Asked Questions ==

= Is it customizable =

Yes!  In the setting screen the code box is provided so you can add, delete, or edit the code as needed.
